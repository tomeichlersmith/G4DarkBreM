Detailed Description

Formal class representing the A' (a dark photon)

This class follows the standard prototype for all G4ParticleDefinitions where a static private member holds onto the single instance of this particle definition to be shared by everyone. In this case, the first call to the accessor function needs to define the A' mass and then all subsequent calls can access it like normal with that configured mass.

This first call to the accessing method should be done in the ConstructParticle function of a physics constructor.

[in]idThe PDG ID number to use for the APrime particle

The default value for the PDG ID is set to 62 and has been arbitrarily chosen from the range defined by 11(c) in the PDG ID numbering scheme, avoiding the already-defined "one-of-a-kind" particles using 39, 41, and 42.
